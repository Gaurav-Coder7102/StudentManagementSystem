
# Student Management System

## Overview

The **Student Management System** is a Java-based application designed to manage the details of students within an academic institution. This system allows administrators to add, update, view, and delete student information, such as personal details, course enrollment, and grades.

## Features

- **Student Registration:** Add new students with personal and academic details.
- **Student Record Management:** Update or delete student information as needed.
- **Course Management:** Assign students to courses and manage their enrollments.
- **Grade Management:** Input and update student grades.
- **Search Functionality:** Easily search for students by name, ID, or course.
- **User Authentication:** Secure login for administrators.

**Technologies Used**

- Programming Language:Java
- Database:MySQL 
- IDE:Eclipse
- Version Control: Git
